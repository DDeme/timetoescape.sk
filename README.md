Public website for escape room business.

## Run Locally

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```

## Tech Stack

Typescript, React, Gatsby, GraphQL, Styled Components, TailwindCSS, React Leaflet
